1 # finderSelect | Jquery & Zepto Plugin
2 
3 [finderSelect](http://github.com/evulse/finderselect) Adds the ability to highlight table rows based on the standard single click, command/ctrl+click, command+drag/ctrl+drag and shift+click methods. This plugin will add a `selected` class to all active rows.
