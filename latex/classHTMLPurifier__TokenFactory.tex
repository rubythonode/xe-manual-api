\subsection{Detailed Description}
Factory for token generation.

\begin{DoxyNote}{Note}
Doing some benchmarking indicates that the new operator is much slower than the clone operator (even discounting the cost of the constructor). This class is for that optimization. Other then that, there's not much point as we don't maintain parallel \hyperlink{classHTMLPurifier__Token}{H\+T\+M\+L\+Purifier\+\_\+\+Token} hierarchies (the main reason why you'd want to use an abstract factory). 
\end{DoxyNote}
