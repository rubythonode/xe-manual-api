\subsection{Detailed Description}
Injects tokens into the document while parsing for well-\/formedness. This enables \char`\"{}formatter-\/like\char`\"{} functionality such as auto-\/paragraphing, smiley-\/ification and linkification to take place.

A note on how handlers create changes; this is done by assigning a new value to the \$token reference. These values can take a variety of forms and are best described H\+T\+M\+L\+Purifier\+\_\+\+Strategy\+\_\+\+Make\+Well\+Formed-\/$>$process\+Token() documentation.
