\subsection{Detailed Description}
Transforms F\+O\+N\+T tags to the proper form (S\+P\+A\+N with C\+S\+S styling)

This transformation takes the three proprietary attributes of F\+O\+N\+T and transforms them into their corresponding C\+S\+S attributes. These are color, face, and size.

\begin{DoxyNote}{Note}
Size is an interesting case because it doesn't map cleanly to C\+S\+S. Thanks to \href{http://style.cleverchimp.com/font_size_intervals/altintervals.html}{\tt http\+://style.\+cleverchimp.\+com/font\+\_\+size\+\_\+intervals/altintervals.\+html} for reasonable mappings. 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
This doesn't work completely correctly; specifically, this Tag\+Transform operates before well-\/formedness is enforced, so the \char`\"{}active formatting elements\char`\"{} algorithm doesn't get applied. 
\end{DoxyWarning}
