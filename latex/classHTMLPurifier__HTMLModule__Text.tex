\subsection{Detailed Description}
X\+H\+T\+M\+L 1.\+1 Text Module, defines basic text containers. Core Module. \begin{DoxyNote}{Note}
In the normative X\+M\+L Schema specification, this module is further abstracted into the following modules\+:
\begin{DoxyItemize}
\item Block Phrasal (address, blockquote, pre, h1, h2, h3, h4, h5, h6)
\item Block Structural (div, p)
\item Inline Phrasal (abbr, acronym, cite, code, dfn, em, kbd, q, samp, strong, var)
\item Inline Structural (br, span) This module, functionally, does not distinguish between these sub-\/modules, but the code is internally structured to reflect these distinctions. 
\end{DoxyItemize}
\end{DoxyNote}
