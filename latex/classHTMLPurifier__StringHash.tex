\subsection{Detailed Description}
This is in almost every respect equivalent to an array except that it keeps track of which keys were accessed.

\begin{DoxyWarning}{Warning}
For the sake of backwards compatibility with early versions of P\+H\+P 5, you must not use the \$hash\mbox{[}\$key\mbox{]} syntax; if you do our version of offset\+Get is never called.
