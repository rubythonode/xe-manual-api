\subsection{Detailed Description}
Fluent interface for validating the contents of member variables. This should be immutable. See \hyperlink{classHTMLPurifier__ConfigSchema__Validator}{H\+T\+M\+L\+Purifier\+\_\+\+Config\+Schema\+\_\+\+Validator} for use-\/cases. We name this an 'atom' because it's O\+N\+L\+Y for validations that are independent and usually scalar.
