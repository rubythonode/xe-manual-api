\subsection{Detailed Description}
Parses string hash files. File format is as such\+: \begin{DoxyVerb} DefaultKeyValue
 KEY: Value
 KEY2: Value2
 --MULTILINE-KEY--
 Multiline
 value.
\end{DoxyVerb}


Which would output something similar to\+: \begin{DoxyVerb} array(
     'ID' => 'DefaultKeyValue',
     'KEY' => 'Value',
     'KEY2' => 'Value2',
     'MULTILINE-KEY' => "Multiline\nvalue.\n",
 )
\end{DoxyVerb}


We use this as an easy to use file-\/format for configuration schema files, but the class itself is usage agnostic.

You can use -\/--- to forcibly terminate parsing of a single string-\/hash; this marker is used in multi string-\/hashes to delimit boundaries.
