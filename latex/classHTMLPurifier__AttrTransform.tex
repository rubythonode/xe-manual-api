\subsection{Detailed Description}
Processes an entire attribute array for corrections needing multiple values.

Occasionally, a certain attribute will need to be removed and popped onto another value. Instead of creating a complex return syntax for \hyperlink{classHTMLPurifier__AttrDef}{H\+T\+M\+L\+Purifier\+\_\+\+Attr\+Def}, we just pass the whole attribute array to a specialized object and have that do the special work. That is the family of \hyperlink{classHTMLPurifier__AttrTransform}{H\+T\+M\+L\+Purifier\+\_\+\+Attr\+Transform}.

An attribute transformation can be assigned to run before or after \hyperlink{classHTMLPurifier__AttrDef}{H\+T\+M\+L\+Purifier\+\_\+\+Attr\+Def} validation. See \hyperlink{classHTMLPurifier__HTMLDefinition}{H\+T\+M\+L\+Purifier\+\_\+\+H\+T\+M\+L\+Definition} for more details.
